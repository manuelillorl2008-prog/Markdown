# Guía de instalación de Python

## Descripción

Este proyecto contiene una documentación creada utilizando Markdown como parte de una práctica de investigación y aplicación del lenguaje.

La documentación explica el proceso de instalación de Python, incluyendo requisitos, pasos de instalación, comandos básicos y recursos útiles.

## Contenido

El proyecto incluye:

- Archivo principal en formato Markdown.
- Imágenes utilizadas en la documentación.
- Ejemplos de sintaxis Markdown.
- Organización de recursos mediante carpetas.
